1. **まず確認すること**

   - 出力可能な項目

2. **メーカーに確認すること**
   - 「会計データをCSV形式で出力できますか？」
   - 「出力できる項目は何ですか？」
   - 「出力フォーマットはカスタマイズできますか？」

3. **試してみること**
   - まず月次で手動CSV出力
   - サンプルデータでインポートテスト
   - 問題なければ定期運用へ
