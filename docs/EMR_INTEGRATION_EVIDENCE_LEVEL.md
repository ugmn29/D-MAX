## エビデンスレベルの定義

- 🟢 **Level A（確実）**: 公式サイト・公式ドキュメントで確認済み
- 🟡 **Level B（推定）**: 業界情報・ユーザー報告に基づく推定
- 🔴 **Level C（不明）**: 確認できず、推測のみ
